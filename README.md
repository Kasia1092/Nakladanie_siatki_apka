# Apka do nakładania siatki na JPG/PNG

Wersja v4.

## Co dodano

- wyśrodkowanie siatki osobno dla każdego pomieszczenia,
- ręczne ustawianie środka kratki osobno dla każdego pomieszczenia,
- numerowanie kratek osobno dla każdego pomieszczenia,
- ustawienie prefixu, np. P, A, K,
- ustawienie numeru od i numeru do,
- ustawienie pozycji napisu w kratce: góra/lewo/środek/prawo/dół,
- ustawienie koloru, rozmiaru, transparentności i pogrubienia numerów,
- nadal działa wybór koloru, wymiarów, grubości i transparentności siatki.

## Jak używać

1. Wgraj JPG/PNG.
2. Kliknij dwa punkty do skali.
3. Wpisz długość odcinka i zapisz skalę.
4. Zaznacz wszystkie narożniki pomieszczenia.
5. Zamknij pomieszczenie.
6. W panelu „Ustawienia osobno dla każdego pomieszczenia” ustaw siatkę i numerację.
7. Kliknij „Pokaż podgląd”.
8. Pobierz PNG.
