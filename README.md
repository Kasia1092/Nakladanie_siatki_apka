# Siatka 1×1 m — prosta wersja

Bardzo prosta statyczna aplikacja do nakładania siatki 1×1 m na pokoje/korytarze na obrazie JPG/PNG.

## Jak działa

1. Wgraj obraz.
2. Kliknij dwa końce znanego odcinka.
3. Wpisz długość tego odcinka w cm albo m.
4. Kliknij „Zapisz skalę”.
5. Klikaj narożniki pokoju/korytarza.
6. Kliknij „Zamknij pokój”.
7. Powtórz dla kolejnych pomieszczeń.
8. Kliknij „Generuj podgląd”.
9. Kliknij „Pobierz PNG”.
