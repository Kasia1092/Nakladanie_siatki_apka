# Apka do nakładania siatki na JPG/PNG

Wersja v8.

## Zmiany

- Usunięto informację o wersji z nagłówka strony.
- Domyślny kolor siatki to czerwony `#ff0000`.
- Domyślny kolor napisów/numerów to czerwony `#ff0000`.
- Dodano przyciski widoku nad obrazem:
  - `− Pomniejsz`,
  - `Dopasuj widok`,
  - `+ Powiększ`,
  - `Przesuń widok`.
- Zoom i przesuwanie działają tylko na widok roboczy — nie zmieniają zapisanych punktów ani eksportu PNG.
- Kropki punktów zostały zastąpione plusami, żeby precyzyjniej widzieć miejsce kliknięcia.

## Jak używać zoomu

- Kliknij `+ Powiększ`, żeby przybliżyć rzut.
- Kliknij `− Pomniejsz`, żeby oddalić.
- Kliknij `Dopasuj widok`, żeby wrócić do widoku całego obrazu.
- Kliknij `Przesuń widok: wyłączone`, żeby włączyć przesuwanie obrazu.
- W trybie przesuwania przeciągasz obraz myszką i przypadkowo nie dodajesz punktów.
- Wyłącz `Przesuń widok`, żeby dalej zaznaczać punkty, obszary albo kratki.
