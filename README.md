# Apka do nakładania siatki na JPG/PNG

Wersja v5.

## Co dodano

- osobne ustawienia siatki dla każdego pomieszczenia:
  - szerokość kratki,
  - wysokość kratki,
  - kolor siatki,
  - grubość siatki,
  - transparentność siatki,
- ustawienia domyślne dla nowych pomieszczeń,
- wyśrodkowanie siatki osobno dla każdego pomieszczenia,
- ręczne ustawianie środka kratki osobno dla każdego pomieszczenia,
- numerowanie kratek osobno dla każdego pomieszczenia,
- prefix, numer od/do, pozycja napisu w kratce,
- kolor, rozmiar, transparentność i pogrubienie numerów.

## Jak używać

1. Wgraj JPG/PNG.
2. Kliknij dwa punkty do skali.
3. Wpisz długość odcinka i zapisz skalę.
4. Zaznacz wszystkie narożniki pomieszczenia.
5. Kliknij „Zamknij pomieszczenie”.
6. W panelu „Opcje pomieszczeń” ustaw osobno siatkę, centrowanie i numerację.
7. Kliknij „Pokaż podgląd”.
8. Pobierz PNG.
